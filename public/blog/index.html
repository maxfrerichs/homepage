Why I used a static site generator instead of a CMS (or WordPress) for my homepage

2022-08-25
As you may already know, I’m working mostly with Content Management Systems at my job, so why I’m using a static site generator I’ve never worked before with? The answer is pretty simple: Because I don’t need most of the funcionality a full-featured CMS has to offer. Thinking about TYPO3, this CMS has not only it’s own configuration meta-language called TypoScript, it also features it’s own templating engine, it’s own APIs and even a complete Domain-driven design MVC framework named Extbase including everything needed for developing large web apps.
